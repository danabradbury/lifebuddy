## Kiosk / doorway view

- **Goal**: create an unavoidable, glanceable Focus view on a wall-mounted screen.

### Software

- Special route `/kiosk` in the web app.
- Shows Focus (habits due today + due/overdue tasks) with larger text and minimal chrome.
- Auto-refresh on a timer so data stays current.

### Hardware (later)

- Tablet or Raspberry Pi connected to a monitor near the door.
- Browser in kiosk mode pointed at `/kiosk`.
- Optional presence detection to wake the display when someone walks by.

This document will expand as we refine the UX and hardware setup.
